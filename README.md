# Personal Website with Guestbook
Full stack web app using NestJS, React, and Supabase
---

# Stack

| Layer      | Technology                |
|------------|---------------------------|
| Frontend   | React + Vite              |
| Backend    | NestJS                    |
| Database   | Supabase (PostgreSQL)     |
| Deployment | Vercel (Frontend), Render (Backend) |
